## Genel Bakış
**AntiWall**, eğitim amaçlı hazırlanmış bir demo antivirüs uygulamasıdır. Bu yazılım **gerçek tehdit temizleme veya sistem dosyalarına müdahale etmez**, yalnızca kullanıcı arayüzü ve sahte tarama özellikleri ile çalışır.  

Bu proje okul projeleri ve deneme amaçlı kullanılabilir.

---

## Özellikler
- Basit ve kullanıcı dostu arayüz
- Dosyalarda Virüs tarama özelliği
- Dosya bilgisi görüntüleme (meta veriler)
- Demo amaçlı güvenli uygulama
- Kurulum sonrası masaüstü kısayolu oluşturma

---

## Kurulum  
ilk olarak SAĞ ÜST köşede "CODE" butonuna basın. Açılan menüden "Download ZIP" yazan butona tıklayın. Tıkladıktan sonra bir indirme işlemi başlayacaktır. İndirme işlemi bittikten sonra indirilen Genel-Projeler-main.zip dosyasını açın. dosyanın içindeki AntiWallSetup.part1.rar dosyasına girin. girdikten sonra AntiWallSetup.exe uygulamasını çalıştırın. Kurulum adımlarını tamamladıktan sonra uygulamayı kullanabileceksiniz

---

- Dosyanın güvenli olduğundan emin olmak için SHA-256 kontrolü veya dosya boyutunu doğrulayabilirsiniz.

## Overview (English)
**AntiWall** is a demo antivirus application designed for educational purposes. This software **does not remove real threats or modify system files**, it only simulates scanning and provides a user interface for demonstration purposes.  

It is suitable for school projects and experimental use.

---

## Features
- Simple and user-friendly interface
- Virus scanning simulation for files
- View file information (metadata)
- Safe demo application
- Creates a desktop shortcut after installation

---

## Installation
First, click the "CODE" button in the top-right corner. From the dropdown menu, select "Download ZIP". After downloading, open the `Genel-Projeler-main.zip` file. Inside, locate `AntiWallSetup.part1.rar` and open it. Then, run `AntiWallSetup.exe` and follow the installation steps. After installation, you can start using the application.

---

## Security Notes
- The demo version does not remove real threats.
- You may see Windows SmartScreen or antivirus warnings during installation.
- To ensure the file is safe, verify its SHA-256 hash or file size.
